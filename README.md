# 📝 ระบบบันทึกคะแนนความประพฤตินักเรียน (Student Behavior Management System)

แอปพลิเคชันบนเว็บ (Web Application) สำหรับจัดการและบันทึกคะแนนความประพฤติของนักเรียน ออกแบบมาเพื่อช่วยอำนวยความสะดวกให้คณะครูประจำชั้นและครูผู้สอนในการตัด/เพิ่มคะแนนอย่างเป็นระบบ รวดเร็ว และประเมินผลได้ทันที 

พัฒนาในรูปแบบ **PWA (Progressive Web App)** สามารถกดติดตั้งลงบนหน้าจอโฮมของสมาร์ทโฟนได้ทั้ง iOS และ Android โดยใช้ **Google Sheets เป็นฐานข้อมูลหลัก** ซึ่งฟรี 100% และไม่ต้องตั้งค่าเซิร์ฟเวอร์ให้ยุ่งยาก

---

## ✨ คุณสมบัติเด่น (Features)

- 📱 **PWA Ready:** ติดตั้งเป็นแอปบนมือถือ ใช้งานได้ลื่นไหลเหมือน Native App
- ⚡ **Fast & Offline-Ready UX:** โหลดรายชื่อนักเรียนเพียงครั้งเดียว ใช้งานหน้าจัดการคะแนนได้ต่อเนื่องโดยไม่ต้องรีเฟรช
- 👥 **Batch Processing:** หักคะแนนโหมดรวดเร็ว เลือกนักเรียนหลายคน และข้อหาความผิดหลายรายการได้พร้อมกันในครั้งเดียว
- 📊 **Smart Dashboard & Leaderboard:** หน้าต่างสรุปสถิตินักเรียนที่ถูกหักคะแนนสูงสุด (เฝ้าระวัง), นักเรียนคะแนนสูงสุด และกราฟความถี่ความผิด
- 🖨️ **Export Slip:** ระบบสร้าง "สลิปรายงานความประพฤติ" แบบพรีเมียม พร้อม QR Code สามารถกดบันทึกเป็นรูปภาพส่งให้ผู้ปกครองได้ทันที
- 🎨 **Customizable:** สามารถเปลี่ยนโลโก้โรงเรียน ภาพพื้นหลัง และธีมสีของแอปพลิเคชันได้จากหน้าตั้งค่า
- 🛡️ **Role Management:** มีรหัสผ่าน (PIN) ป้องกันการหักคะแนน และระบบจัดการสถานะโดยครูประจำชั้น (เช่น เชิญผู้ปกครอง, พักการเรียน, รีเซ็ตคะแนน)

---

## 🛠️ เทคโนโลยีที่ใช้ (Tech Stack)

- **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
- **Styling:** Tailwind CSS (ผ่าน CDN)
- **Icons & Fonts:** FontAwesome 6, Google Fonts (Sarabun, Amiri)
- **Backend / Database:** Google Apps Script (GAS) & Google Sheets
- **Libraries:** `html2canvas.js` (สำหรับบันทึกภาพสลิป), `qrcode.js`

## 🚀 วิธีติดตั้งและใช้งานสำหรับโรงเรียนอื่น

หากสนใจนำระบบนี้ไปใช้กับโรงเรียนของคุณ สามารถทำตาม 3 ขั้นตอนง่ายๆ ดังนี้:

### 1. ทำสำเนาฐานข้อมูล (Google Sheets Template)
- คลิกที่ลิงก์นี้: [คัดลอกแม่แบบฐานข้อมูล](https://docs.google.com/spreadsheets/d/11Y4wRTtI4ZI5lwYslNHJVAMlY4Fbx_gukxTez5_CpPg/copy)
- ระบบจะทำสำเนาไฟล์เข้าสู่ Google Drive ของคุณโดยอัตโนมัติ
- ไปที่แท็บ `Students` ลบข้อมูลตัวอย่างออก และกรอกรายชื่อนักเรียนของคุณ:
  - **คอลัมน์ A (Name):** ชื่อ-สกุล
  - **คอลัมน์ B (Room):** ชั้น/ห้องเรียน (เช่น 1/1, 2/1) *แนะนำให้ปรับคอลัมน์เป็นรูปแบบข้อความ (Plain Text) ก่อนพิมพ์*
  - **คอลัมน์ C (Score):** ใส่คะแนนเริ่มต้น `100` ทุกคน
  - *คอลัมน์ D, E, F ปล่อยว่างไว้*

### 2. เปิดใช้งานระบบ API (Deploy Apps Script)
1. ในไฟล์ Google Sheets ที่ทำสำเนาไว้ ไปที่เมนู **ส่วนขยาย (Extensions)** > **Apps Script**
2. คลิกปุ่มสีน้ำเงินมุมขวาบน **การทำให้ใช้งานได้ (Deploy)** > **การทำให้ใช้งานได้รายการใหม่ (New deployment)**
3. คลิกฟันเฟือง ⚙️ ด้านซ้าย เลือกประเภท **เว็บแอป (Web app)**
4. ตรงส่วน สิทธิ์การเข้าถึง (Who has access) ให้เลือกเป็น **"ทุกคน (Anyone)"**
5. กด **Deploy** (หากมีการขอสิทธิ์ ให้ล็อกอินอีเมลและกดยอมรับ Advanced > Allow)
6. **คัดลอก URL เว็บแอป** (ที่ลงท้ายด้วย `/exec`) เก็บเอาไว้

### 3. เชื่อมต่อแอปพลิเคชัน
1. เข้าไปที่แอปพลิเคชันนี้ (หน้าเว็บที่รันไฟล์ `index.html`)
2. ไปที่เมนู **ตั้งค่าแอปพลิเคชัน**
3. วาง **URL เว็บแอป** ที่ได้จากขั้นตอนที่ 2 ลงในช่อง "ลิงก์ฐานข้อมูล (Google Apps Script URL)"
4. เปลี่ยนชื่อโรงเรียน และโลโก้ตามต้องการ แล้วกด **บันทึกการตั้งค่า**

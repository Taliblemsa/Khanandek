ข้อมูลระบบ

📝 ระบบบันทึกคะแนนความประพฤตินักเรียน

แอปพลิเคชันบนเว็บ (Web Application) สำหรับจัดการและบันทึกคะแนนความประพฤติของนักเรียน ออกแบบมาเพื่อช่วยอำนวยความสะดวกให้คณะครูประจำชั้นและครูผู้สอนในการตัด/เพิ่มคะแนนอย่างเป็นระบบ รวดเร็ว และประเมินผลได้ทันที

พัฒนาในรูปแบบ PWA (Progressive Web App) สามารถกดติดตั้งลงบนหน้าจอโฮมของสมาร์ทโฟนได้ทั้ง iOS และ Android โดยใช้ Google Sheets เป็นฐานข้อมูลหลัก ซึ่งฟรี 100% และไม่ต้องตั้งค่าเซิร์ฟเวอร์ให้ยุ่งยาก

คุณสมบัติเด่น (Features)

  • 📱 PWA Ready: ติดตั้งเป็นแอปบนมือถือ ใช้งานได้ลื่นไหลเหมือน Native App
  • ⚡ Fast & Offline-Ready UX: โหลดรายชื่อนักเรียนเพียงครั้งเดียว ใช้งานได้ต่อเนื่อง
  • 👥 Batch Processing: หักคะแนนโหมดรวดเร็ว เลือกเด็กและข้อหาได้หลายรายการพร้อมกัน
  • 📊 Smart Dashboard: สรุปสถิติ จัดอันดับเฝ้าระวัง และกราฟความถี่
  • 🖨️ Export Slip: สร้าง "สลิปรายงานความประพฤติ" แบบพรีเมียม พร้อม QR Code
  • 🎨 Customizable: เปลี่ยนโลโก้ ภาพพื้นหลัง และธีมสีได้อิสระ
  • 🛡️ Role Management: ล็อกรหัสผ่าน PIN และระบบจัดการสถานะโดยครูประจำชั้น

เทคโนโลยีที่ใช้

  • Frontend: HTML5, CSS3, JavaScript (Vanilla)
  • Styling: Tailwind CSS
  • Icons & Fonts: FontAwesome 6, Sarabun, Amiri
  • Backend/DB: Google Apps Script (GAS) & Google Sheets

วิธีสร้างฐานข้อมูล (สำหรับโรงเรียนอื่น)

1. ทำสำเนาแม่แบบฐานข้อมูล
ลบรายชื่อเดิมออก ใส่ข้อมูล: A (ชื่อสกุล), B (ชั้นเรียน เช่น 1/1), C (คะแนนตั้งต้น 100), ส่วน D-F ปล่อยว่าง
คลิกลิงก์เพื่อทำสำเนาชีต
2. เปิดใช้งาน API (Deploy)
ในชีตไปที่ ส่วนขยาย > Apps Script กด การทำให้ใช้งานได้ > รายการใหม่ เลือก เว็บแอป ตั้งสิทธิ์เป็น "ทุกคน (Anyone)" แล้วกด Deploy
3. เชื่อมต่อระบบ
คัดลอก URL เว็บแอป (ที่ลงท้ายด้วย /exec) มาวางในช่องด้านบน แล้วกดบันทึก
